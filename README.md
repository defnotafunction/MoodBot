# MoodBot
A chatbot that uses a trained model to predict your mood and web scraping to respond with quotes.

## Main Features
- App - Built with `streamlit` for an interactive chat interface
- Quotes - MoodBot's responses consist of a scraped quote that correlates to your mood.
- Guessing - MoodBot guesses your mood by using a Naive Bayes model, which was trained on hundreds of sentences, from `scikit-learn`.

## Technologies Used
- **Python 3.11+**
- **Streamlit**
- **Scikit-learn**
- **Requests**
- **BeautifulSoup**
- **JSON**
